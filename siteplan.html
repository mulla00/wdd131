Site Plan: Mulla Inc

Site Name

mulla-inc Cargo is a professional cargo transportation and Logisitics website providing secure and reliable freight services globally.

Domain availability: mullainc.org

Site Purpose

The purpose of mulla-inc Cargo is to offer a user-friendly platform platform where customers can track shipments, request quotes, and learn about freight shipping services.

Scenarios

  • How can I track my cargo shipment?
  • What shipping services does mulla-inc Cargo offer?

Color Schema

  • Primary Color: #003366 (Headers, Navigation)
  • Accent Color: #ffcc00 (Buttons, Highlights)
  • Background Color: #f5f5f5 (Main Background)

Typography

Primary Font: Montserrat (Used for headings)

Secondary Font: Open Sans (Used for body text)
